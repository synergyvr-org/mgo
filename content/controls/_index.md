Mad God's Overhaul includes several control schemes to choose from.

{{< chapter-heading href="bindings-vrik" >}}
## VRIK

The default controls packaged with VRIK are designed to be better suited to _modded_ Skyrim VR than the vanilla controls.
{{< /chapter-heading >}}

{{< chapter-heading href="bindings-vrsafe" >}}
## VR Safe

A close-to-vanilla scheme that shuffles a few actions (sprint, for one) onto safer inputs.
{{< /chapter-heading >}}

{{< chapter-heading href="bindings-touch" >}}
## Optimized Oculus

Built for Oculus and Meta Touch controllers. It turns the face buttons into hotkeys and the grips into modifiers, opening up a lot of room for mod actions.
{{< /chapter-heading >}}

{{< chapter-heading href="bindings-kvite" >}}
## Kvite

One of the control schemes from the FUS modlists, by Kvitekvist. Uses vanilla bindings within menus.
{{< /chapter-heading >}}

{{< chapter-heading href="bindings-cangar" >}}
## Cangar

Another scheme from the FUS modlists, by Cangar, that leans on gestures&mdash;there's no dedicated shout button by default.
{{< /chapter-heading >}}
