{{< nexus 90557 >}}VRAMr{{< /nexus >}} doesn't change any in-game settings. It optimizes your textures ahead of time, generating downscaled, recompressed copies of the thousands of textures in the list, which your GPU can store and decode more efficiently. The result is lower VRAM use and fewer stutters, which matters a great deal if your card is tight on VRAM. VRAMr is highly recommended for any video card with under 16GB of memory. (If you're right at 16GB, you're probably fine.)

MGO includes the VRAMr tool, but you'll need to run it yourself to generate the optimized textures. That output takes up a lot of space, so it isn't bundled with the list. (The download is big enough already.) If you're fighting VRAM limits, it's well worth the time and effort. The {{< nexus 90557 >}}VRAMr Nexus page{{< /nexus >}} has instructions, and there are plenty of video tutorials out there as well.

Check out the {{< youtube "Se6oAvmtaBE?si=nC7mhPYHiOehJdt5&t=197"  >}}VRAMr Installation and User Video{{< /nexus >}}. Since the tool is already installed with MGO, you can skip to running it.
