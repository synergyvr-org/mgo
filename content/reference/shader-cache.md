If you have an NVIDIA GPU, you should set the shader cache to 10GB or more before playing Mad God's Overhaul. (AMD GPUs do no currently have an equivalent setting.)

## Without screenshots...

1. Open the NVIDIA Control Panel
2. Choose {{< btn-inline >}}Manage 3D settings{{</ btn-inline >}} on the left.
3. Under _Global Settings_ on the right, set {{< btn-inline >}}Shader Cache Size{{</ btn-inline >}} to 10 GB or more.
