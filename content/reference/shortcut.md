The MGO desktop shortcut that Wabbajack creates can't be pinned to the start menu (for reasons), so it's not ideal. What follows is my preferred method.

1. Open your MGO installation folder in File Explorer (you set this in Wabbajack).
2. Right-click {{< file >}}ModOrganizer.exe{{< /file>}}, and choose {{< btn-inline >}}<i class="fa-solid fa-thumbtack"></i> Pin to Start{{< /btn-inline>}}.
3. Open the {{< btn-inline >}}<i class="fa-brands fa-windows"></i> Start{{< /btn-inline>}} menu.
4. Right-click the pinned {{< file >}}ModOrganizer{{< /file>}} shortcut, and choose {{< btn-inline >}}<i class="fa fa-folder-open"></i> Open file location{{< /btn-inline>}}.
5. To choose a different icon, right-click the {{< file >}}ModOrganizer{{< /file>}} shortcut in the new File Explorer window, and choose {{< btn-inline >}}<i class="fa fa-wrench"></i> Properties{{< /btn-inline>}}.
6. Assign a more interesting icon&mdash;like this one!

7. Rename the shortcut to something catchy, like _Mad God's Overhaul_.
8. Open the {{< btn-inline >}}<i class="fa-brands fa-windows"></i> Start{{< /btn-inline>}} menu and behold your pinned shortcut in all its glory!

{{< aside type="btw" title="By the way..." >}}
You can open File Explorer any time with <kbd><i class="fa-brands fa-windows"></i></kbd> + <kbd>E</kbd>. It's my favorite keyboard shortcut in all of Windows. (My second favorite is <kbd><i class="fa-brands fa-windows"></i></kbd> + <kbd>R</kbd>, not that you asked.)
